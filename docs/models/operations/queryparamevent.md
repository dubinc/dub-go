# QueryParamEvent

The type of event to retrieve analytics for. Defaults to 'clicks'.

## Example Usage

```go
import (
	"github.com/dubinc/dub-go/models/operations"
)

value := operations.QueryParamEventClicks
```

| Name                    | Value                   |
| ----------------------- | ----------------------- |
| `QueryParamEventClicks` | clicks                  |
