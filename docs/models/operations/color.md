# Color

The color of the tag. If not provided, a random color will be used from the list: red, yellow, green, blue, purple, pink, brown.


## Values

| Name          | Value         |
| ------------- | ------------- |
| `ColorRed`    | red           |
| `ColorYellow` | yellow        |
| `ColorGreen`  | green         |
| `ColorBlue`   | blue          |
| `ColorPurple` | purple        |
| `ColorPink`   | pink          |
| `ColorBrown`  | brown         |
