# SortOrder

The sort order. The default is `desc`.

## Example Usage

```go
import (
	"github.com/dubinc/dub-go/models/operations"
)

value := operations.SortOrderAsc
```


## Values

| Name            | Value           |
| --------------- | --------------- |
| `SortOrderAsc`  | asc             |
| `SortOrderDesc` | desc            |
