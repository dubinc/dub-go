# Event

The type of event to retrieve analytics for. Defaults to `clicks`.

## Example Usage

```go
import (
	"github.com/dubinc/dub-go/models/operations"
)

value := operations.EventClicks
```


## Values

| Name             | Value            |
| ---------------- | ---------------- |
| `EventClicks`    | clicks           |
| `EventLeads`     | leads            |
| `EventSales`     | sales            |
| `EventComposite` | composite        |
