# Level

The level of error correction to use for the QR code. Defaults to `L` if not provided.

## Example Usage

```go
import (
	"github.com/dubinc/dub-go/models/operations"
)

value := operations.LevelL
```


## Values

| Name     | Value    |
| -------- | -------- |
| `LevelL` | L        |
| `LevelM` | M        |
| `LevelQ` | Q        |
| `LevelH` | H        |
