# SortBy

The field to sort the links by. The default is `createdAt`.


## Values

| Name                | Value               |
| ------------------- | ------------------- |
| `SortByCreatedAt`   | createdAt           |
| `SortByClicks`      | clicks              |
| `SortBySaleAmount`  | saleAmount          |
| `SortByLastClicked` | lastClicked         |
