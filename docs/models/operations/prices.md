# Prices

Price details for the domain. Will be null if the domain is not available.


## Fields

| Field                                         | Type                                          | Required                                      | Description                                   |
| --------------------------------------------- | --------------------------------------------- | --------------------------------------------- | --------------------------------------------- |
| `Registration`                                | `*float64`                                    | :heavy_check_mark:                            | The domain's registration price in USD cents. |
| `Renewal`                                     | `*float64`                                    | :heavy_check_mark:                            | The domain's renewal price in USD cents.      |
