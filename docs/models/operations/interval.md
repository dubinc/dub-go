# Interval

The interval to retrieve analytics for. If undefined, defaults to 24h.


## Values

| Name                    | Value                   |
| ----------------------- | ----------------------- |
| `IntervalTwentyFourh`   | 24h                     |
| `IntervalSevend`        | 7d                      |
| `IntervalThirtyd`       | 30d                     |
| `IntervalNinetyd`       | 90d                     |
| `IntervalYtd`           | ytd                     |
| `IntervalOney`          | 1y                      |
| `IntervalAll`           | all                     |
| `IntervalAllUnfiltered` | all_unfiltered          |
